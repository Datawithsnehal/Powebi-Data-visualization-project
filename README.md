1.Project Title Sales Performance Dashboard using Power BI | Revenue, Profit & Growth Insights This project analyzes sales data using Power BI to track revenue, profit, and business growth. It includes time intelligence measures like YTD, MTD, and trend analysis to provide meaningful insights.

2.Short description and perpose This project is a Power BI dashboard that analyzes sales data to track revenue, profit, and overall business performance. It uses time intelligence features like YTD, MTD, and growth trends to provide clear and meaningful insights. The purpose of this project is to understand business performance by analyzing sales data. It helps identify revenue trends, profit levels, top-performing products, and growth over time. This dashboard supports better decision-making and improves business strategies.

3.Tech stack

Power BI Desktop – Data visualization and dashboard creation
Microsoft Excel – Data source and data preparation
DAX (Data Analysis Expressions) – Calculations and measures (YTD, MTD, Profit, etc.)
Power Query – Data cleaning and transformation
Data Modeling – Relationships, calendar table, and schema design

5.Dataset Info(Electronics Dataset) This project uses an electronics sales dataset that contains information about orders, products, customers, and financial details. The dataset includes various electronic products such as laptops, mobile devices, and accessories.

Columns in the dataset:

Order_Number – Unique ID for each order
State_Code – Location identifier of the order
Customer_Name – Name of the customer
Order_Date – Date when the order was placed
Status – Order status (Delivered, Pending, etc.)
Product – Name of the electronic product
Category – Product category (e.g., Laptop, Accessories)
Brand – Brand of the product (e.g., Dell, HP, Asus)
Cost – Cost price per unit
Sales – Selling price per unit
Quantity – Number of units sold
Total_Cost – Total cost (Cost × Quantity)
Total_Sales – Total sales (Sales × Quantity)
Assigned Supervisor – Person responsible for managing the order
Year – Extracted from Order_Date for analysis
Profit – Calculated as Total_Sales – Total_Cost
This dataset is used to analyze sales performance, profit, and growth trends in the electronics business using Power BI.
